## Installation

### macOS
1. Download the `.dmg` file for your Mac
2. Open the DMG and drag **HeyApex** to your Applications folder
3. Launch HeyApex from Applications
4. The app is code-signed and notarized by Apple — no security warnings

### Windows
1. Download the `.exe` installer
2. Run the installer and follow the prompts
3. Launch HeyApex from your Start menu
